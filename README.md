Welcome to the official repository for **The NFL Legend Experience** website. 

This website supports the YouTube show hosted by 2X Super Bowl Champion Gary Clark and co-host Chad Morton. The platform serves as a hub where NFL Legends share unforgettable stories and turn those moments into real rewards for real fans.

## Features & Pages
The website is built following a fan-first, story-driven model and includes the following key pages:
- **Home:** An immersive introduction to the show, hosts, and fan rewards.
- **About the Show:** The mission, the heart of the show, and what fans can expect.
- **Meet the Hosts:** Bios and backgrounds for Gary Clark and Chad Morton.
- **Watch the Show:** A library of full episodes and highlight clips embedded directly from YouTube.
- **NFL Legend Experiences (Fan Rewards):** Information on how fans can share their stories and get selected for real, once-in-a-lifetime rewards.
- **Legends Circle (Membership):** Details about the inner fan community, exclusive perks, and early access.
- **Partner & Sponsor:** A breakdown of all available corporate sponsorship tiers (Black, Platinum, Gold, Silver, Bronze, All-Madden) to support the show.
- **Contact:** An easy-to-use form for general inquiries, media, bookings, and fan stories.

## Technology Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Routing:** React Router DOM

## Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/aashath0317/gary_clark_02_20_2026.git
   ```
2. Navigate to the project directory:
   ```bash
   cd gary_clark_02_20_2026
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`.
